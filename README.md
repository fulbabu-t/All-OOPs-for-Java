# 📘 Java Object-Oriented Programming (OOP) Concepts Implementation


## 📌 Project Summary

A hands-on Java project focused on mastering Object-Oriented Programming (OOP). This repository contains multiple examples and class implementations that demonstrate how OOP concepts work in real-world scenarios, making it a useful resource for beginners and intermediate developers.


## 📂 Project Structure (OOP Concepts)

This project contains multiple compiled Java classes, each representing different Object-Oriented Programming (OOP) concepts.

### 📁 Files Overview

- **A.class, B.class, C.class, D.class**  
  Basic classes used to demonstrate fundamental concepts like class creation, objects, and method usage.

- **Demo.class**  
  Entry point class used to test and execute different functionalities.

- **OOPs.class**  
  Main class combining multiple OOP concepts in one place.

- **OOPs$A.class, OOPs$B.class, OOPs$C.class**  
  Inner classes used to demonstrate nested class behavior and relationships.

- **Test.class**  
  Contains test cases or driver code to validate logic.

- **X.class, Y.class**  
  Likely used to demonstrate inheritance or polymorphism relationships.

---

### 🧠 OOP Concepts Covered

- **Encapsulation**  
  Wrapping data and methods into a single unit (class)

- **Inheritance**  
  One class acquiring properties of another (e.g., X → Y)

- **Polymorphism**  
  Same method behaving differently (overloading & overriding)

- **Abstraction**  
  Hiding implementation details and showing only functionality

- **Classes & Objects**  
  Basic building blocks of Java programs

- **Method Overloading & Overriding**  
  Compile-time and runtime polymorphism

- **Inner Classes**  
  Classes defined inside another class (e.g., OOPs$A)
